Configuration
Ganache setup:
1. Open Ganache app → New Workspace → add truffle-config.js
2. Set RPC Server to HTTP://127.0.0.1:8545
3. Run npm run migrate in terminal
4. Paste the deployed contract address below
📄
truffle-config.js — key settings
networks: {
  development: {
    host: "127.0.0.1",
    port: 8545,          ← Ganache GUI port
    network_id: "5777",  ← Ganache GUI network ID
  }
}
In Ganache app → Settings → Server — make sure port matches 8545 and network ID matches 5777.
