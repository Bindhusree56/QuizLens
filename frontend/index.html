Quick Setup Guide

1. Start Ganache: Run npx hardhat node or open Ganache GUI

2. Deploy Contract: Run npx hardhat run scripts/deploy.js --network localhost

3. Copy Address: The deployment script outputs the contract address

4. Paste Here: Enter the contract address above and click "Save Config"

5. Start NLP: Run cd nlp && source venv/bin/activate && uvicorn app:app --reload
